# Disk Scheduling Algorithm Visualization

A web-based project that visualizes various **Disk Scheduling Algorithms** used in Operating Systems.  
The application is built using **HTML, CSS, and JavaScript**, where JavaScript handles the algorithm logic and animations for disk head movement.
